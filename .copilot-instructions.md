\# WorkTrack Pro Architecture Context

\- Stack: Java 17, Spring Boot 3.2.5, PostgreSQL 15, Angular 20.

\- Architecture: Distributed Microservices Mesh (Docker Compose).

\- Services: Eureka (Discovery), Config-Server, API-Gateway (Proxy), AI-Agent, Task-Service, Employee-Service, Leave-Service.

\- Networking: All inter-service calls MUST use container names (e.g., http://task-service:8082), never 'localhost'.

\- Database: All services connect to 'host.docker.internal:5432'.

\- Security: Stateless JWT-based authentication.

\- Resilience: All downstream REST calls are wrapped in Resilience4j circuit breakers.

\- Goal: Maintain loose coupling and fault-tolerant service communication.
